- 알고리즘 풀이

  - ```python
    T = int(input())
    for tc in range(1, T+1):
        N = int(input())
        data = [[0 for _ in range(N)] for _ in range(N)]
        for i in range(N):
            data[i] = list(map(int, input()))
        ans = 0
    
        for y in range(N):
            ans += data[y][N//2]
            if 1<=y and y <= (N//2):
                for dy in range(1, y+1):
                    ans += data[y][N//2-dy] + data[y][N//2+dy]
                    # print(y, dy, N//2-dy, N//2+dy)
            if (N//2) < y and y < N:
                for dy in range(1, N-y):
                    ans += data[y][N//2-dy] + data[y][N//2+dy]
                    # print(y, dy, N//2-dy, N//2+dy)
    
        print('#{} {}'.format(tc, ans))
    ```
